# Chapter 2: The Hardware Stack

This chapter covers the essential hardware components and setup for creating a digital twin workstation.

## Overview

In this chapter, we explore the hardware stack needed for physical AI development, focusing on creating a digital twin environment.

## Topics Covered

- Setting up the "Digital Twin" Workstation
- Introduction to Jetson Orin
- RealSense depth cameras
- Hardware requirements and specifications

## Key Components

:::tip
The **NVIDIA Jetson Orin** provides the computational power needed for AI algorithms on robotics platforms, making it ideal for edge AI applications in humanoid robots.
:::

## Setup Requirements

- NVIDIA Jetson Orin development kit
- Intel RealSense depth camera
- Compatible power supply and cooling
- Appropriate mounting solutions
